# Brand assets

## Faro AI

Para usar o logo PNG (que você enviou no chat) no lugar do SVG inline gerado pelo
componente `FaroLogo`, salve o arquivo aqui:

```
apps/web/public/brand/faro-ai-logo.png
```

Recomendado: 512×512 (transparente). O componente detecta automaticamente quando
o arquivo existe e passa a usá-lo. Sem o arquivo, ele cai pro SVG inline (que é
uma reconstrução estilizada do mesmo motif: farol + estrada).

Versão monocromática branca pra sidebar é gerada automaticamente pelo SVG
quando `monochrome={true}` é passado.
